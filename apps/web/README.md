# ARSY Frucht Website

Това е уебсайтът за ARSY Frucht, изграден с Next.js (App Router).

## Стартиране локално

За да стартирате проекта в режим на разработка:

```bash
npm run dev
```

След това отворете [http://localhost:3000](http://localhost:3000) във вашия браузър.

## Деплоймънт (Vercel)

За Vercel е нужно да зададете **Root Directory** на `apps/web` в Project Settings -> General.
